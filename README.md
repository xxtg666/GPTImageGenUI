# GPT 绘图工作台

一个轻量单体 WebUI：Python 后端代理绘图请求，前端提供文生图、图生图、画廊、提示词库和运行设置。

## 启动

安装依赖：

```powershell
python -m pip install -r requirements.txt
```

```powershell
python app.py
```

默认地址：

```text
http://127.0.0.1:7860
```

也可以指定监听地址和端口：

```powershell
$env:HOST="0.0.0.0"
$env:PORT="7860"
python app.py
```

## 配置

首次启动后进入「设置」页面，填写：

- `后端接口地址`：默认 `https://api.openai.com/v1`
- 使用 NewAPI 时填写你的 NewAPI 基地址，并包含 `/v1`，例如 `https://your-newapi.example.com/v1`
- `模型名`：默认 `gpt-image-1`
- `API Key`：由 Python 后端保存和使用，浏览器不会直接请求上游绘图接口
- `新 WebUI 密码`：设置后访问 WebUI 需要登录

配置文件保存在 `data/config.json`，图库索引和提示词库保存在 `data/db.json`，图片文件保存在 `data/images/`。

## 功能

- 文生图：后端调用 `POST /images/generations`，拼接到设置里的 `/v1` 基地址后即 `/v1/images/generations`
- 图生图：后端调用 `POST /images/edits`，拼接到设置里的 `/v1` 基地址后即 `/v1/images/edits`
- 分辨率：支持常用 OpenAI/NewAPI 尺寸、`auto` 和自定义宽高
- 高级参数：工作台支持 `quality` 和 `style`；生成数量固定为 `1`
- 图生图参数：支持多参考图上传和可选 `mask` 蒙版
- 网页蒙版：图生图上传参考图后可直接在网页中涂抹蒙版；涂抹区域会导出为透明区域 PNG mask，并优先于手动上传的 mask 提交
- 蒙版编辑：关闭涂抹弹窗后可再次打开继续编辑，涂抹覆盖层使用半透明预览
- 画廊：保存所有生成图片、提示词、模型、分辨率和标签
- 画廊操作：显示 `revised_prompt` 作为图片标题，支持复制提示词、下载图片、填充到工作台、搜索过滤；提示词在悬停复制按钮时弹出预览，标题、标签和删除收在卡片下方齿轮弹窗中
- 图生图画廊：会保存第一张参考原图，图生图结果卡片可一键查看原图；大图预览窗口支持下载当前显示的图片
- 画廊布局：使用瀑布流自适应排版，保留不同尺寸图片的实际预览比例
- 提示词库：添加收藏提示词，一键复制或填充到工作台
- 提示词库：支持编辑和删除，删除使用 WebUI 内置确认窗口
- 图生图：切换到图生图模式时分辨率默认使用 `auto`
- 设置页：可配置失败自动重试次数和文生图默认分辨率；工作台高级参数仅保留质量和风格
- 设置页：可配置服务端口，保存后需要重启服务才会生效；`PORT` 环境变量仍优先于配置文件
- 多后端：支持配置多个后端接口，每个后端可单独设置别名、URL、模型名和 API Key；工作台右上角可切换当前活跃后端
- 任务列表：生成请求会作为后台任务并行执行，支持设置失败自动重试次数
- 任务进度：任务卡片显示已用时间，并按设置页中的预期生成时间计算进度条
- 原始报错：任务失败时可展开查看上游 HTTP 状态、JSON 错误或原始文本响应
- 上游请求：使用 `httpx` 发送，启用重定向跟随和 HTTP/2 支持

## 说明

自定义分辨率会原样转发为 `宽x高`。如果上游接口或模型不支持某个尺寸，上游会返回错误，页面会展示后端收到的错误信息。

任务列表保存在当前 Python 进程内，重启服务后会清空任务历史；已成功生成的图片仍会保留在本地画廊中。

NewAPI 前面如果有 Cloudflare 或防盗链策略，后端请求会带上 Cherry Studio 风格的请求头，包括 `HTTP-Referer: https://cherry-ai.com`、`X-Title: Cherry Studio` 和 Electron/Chromium `User-Agent`。
